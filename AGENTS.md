# Agent Guidelines for Dreamscape Tours ZM

This file contains persistent guidelines for the AI assistant and must be loaded for every turn.

## Gallery Image Integrity Rules
- **DO NOT** recreate, substitute, regenerate, or alter any gallery images uploaded by the user or present in the repository.
- Always use the original filenames, paths, and URLs exactly as provided by the user.
- Do not attempt to use AI image generation to beautify, rebuild, or replace original assets.
- Ensure all local image asset paths remain intact and are correctly mapped to `/public` or their respective asset paths to avoid breaking Vite compilation.
